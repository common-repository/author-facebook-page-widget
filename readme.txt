== Description ==

Author Facebook Page Widget is simple and easy to use wordpress plugin to show facebook page feed in your website sidebar. Flexible with lot's of option to enable or disable fb cover photo, call to action button, show friends facepile and most interesting, you can allow your authors to show their page feed on their posts. It will boost their performance and interest in your blog.

Major features of Author Facebook Page Widget:

1. Displays Facebook Page Feed
2. Option to Allow or Disallow, author's to show their facebook page widget in their posts.
3. Option to Enable or Disable call to action button.
4. Enable or Disable Facepile.
5. Facebook script loads asynchronously.
6. Display Page timeline, events, messages tabs.
7. Facebook Page Shortcode.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/author-facebook-page-widget` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Appearance->Widgets->Facebook Page Widget to add widget to sidebar.

= Shortcode Instructions =
Basic Shortcode Example:

`[AuthorFacebookPageFeed href="https://facebook.com/facebook" tabs="timeline"]`

Custom Shortcode Example:

`[AuthorFacebookPageFeed href="https://facebook.com/facebook" width="800" height="400" tabs="timeline, events, messages" show_cta="true" small_header="false" hide_cover="false" show_facepile="true" enable_author_fb_feed="true"]`
